1. Introduction to the System 0:01

The system automates appointment bookings for clinics and doctors.
Reduces the number of WhatsApp messages received for appointments.
 
3. Patient Interaction via WhatsApp 0:13

Patients can initiate contact by sending a message on WhatsApp.
The system automatically replies to the patient's message.
 
3. Available Options for Patients 0:35

Patients receive options in the reply:
Book an appointment
Check clinic timings
Get location
Example: To get the location, the patient can simply type 'location'.
 
4. Receiving Clinic Information 1:23

Upon requesting location or timings, patients receive predefined information from the clinic.
 
5. Booking an Appointment 2:00

Patients can type 'book' or 'book appointment' to initiate the booking process.
 
6. Providing Patient Details 2:14

The system requests the patient's full name for the appointment.
 
7. Selecting a Doctor 2:32

Patients receive a list of available doctors and can select one by typing the corresponding number.
 
8. Choosing Appointment Date and Time 2:43

The system prompts the patient to choose a date and time for the appointment.
 
9. Confirmation of Booking 3:01

Once the patient selects the date and time, the booking is confirmed.
 
10. Cancelling an Appointment 3:20

Patients can cancel appointments by simply typing 'cancel'.
 
11. Admin Panel for Doctors 3:38

Doctors can view and manage all bookings through an admin panel.
 
12. Updating Doctor Availability 4:08

Doctors can edit their schedules and availability from the admin panel.
 
13. Data Security and Storage 4:20

All appointment details are securely stored and can be accessed by the clinic.
 
14. Contact Information 4:34

Interested clinics can contact for more information about implementing the system.
